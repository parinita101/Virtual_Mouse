<h2>AI Virtual Mouse using Hand Gestures</h2>

An AI-based Virtual Mouse that allows users to control the computer mouse using hand gestures captured through a webcam.
This project uses MediaPipe Hand Tracking and OpenCV to detect hand landmarks and perform mouse actions.

<h2>Features</h2>
<ul>
  <li>Real-time hand tracking </li>
  <li>Cursor movement using index finger</li>
  <li>Gesture-based mouse control</li>
  <li>Works with a standard webcam </li>
  <li>Lightweight and fast</li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li>Python </li>
  <li>OpenCV</li>
  <li>MediaPipe</li>
  <li>NumPy</li>
  <li>PyAutoGUI</li>
</ul>

<h2>System Required</h2>
<ul>
  <li>Windows 10/ Windows 11</li>
  <li>Python 3.8 – 3.10 (Recommended: Python 3.10)</li>
  <li>Webcan</li>
</ul>
<p>MediaPipe is not fully compatible with Python 3.11+ on Windows</p>
